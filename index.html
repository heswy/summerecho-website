配置教程

如何获取API Key

夏至点使用硅基流动（SiliconFlow）的 AI 服务，需要您自行申请 API Key

1

注册硅基流动账号

访问硅基流动官网，使用邮箱或手机号注册账号

前往注册
2

创建 API Key

登录后进入「账号管理」→「API 密钥」，点击「新建密钥」

建议给密钥设置一个容易识别的名称，如"夏至点"

3

复制 API Key

创建成功后，点击「复制」按钮获取 API Key（以 sk- 开头）

请妥善保管您的 API Key，不要分享给他人

4

在夏至点中配置

打开夏至点 App，进入「我的」→「AI 设置」，粘贴 API Key

点击「保存」
点击「测试连接」验证是否配置成功
开始使用「观星」AI 功能

关于费用

硅基流动提供免费额度，新用户注册后有一定数量的免费 Token。
具体收费标准请参考硅基流动官网。正常使用情况下，费用非常低廉。
