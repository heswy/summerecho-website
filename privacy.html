隐私政策

最后更新日期：2026年3月8日

夏至点（SummerEcho）尊重并保护所有用户的隐私权。本隐私政策将帮助您了解我们如何收集、使用和保护您的个人信息。

1. 数据存储方式

所有数据本地存储：夏至点是一款纯本地运行的应用程序，您的所有数据（包括联系人信息、互动记录、关系连接等）均存储在您的设备本地，不会上传到任何服务器。

我们无权访问、查看或处理您设备上的任何数据。您对自己的数据拥有完全的所有权和控制权。

2. 数据收集说明

我们不收集以下信息：

  • 个人身份信息（姓名、电话、地址等）
  • 设备识别信息
  • 使用行为数据
  • 位置信息
  • 任何上传到云端的数据

3. AI 服务使用

夏至点的「观星」功能使用硅基流动（SiliconFlow）提供的 AI 服务。当您使用此功能时：

  • 您需要自行申请 API Key 并配置到应用中
  • 相关数据会发送到硅基流动的服务器进行处理
  • 硅基流动的隐私政策适用于此部分服务
  • 建议您查阅硅基流动的隐私政策以了解更多信息

4. 数据安全

虽然数据存储在本地，我们仍建议您：

  • 定期导出备份，防止数据丢失
  • 妥善保管您的设备，防止他人未经授权访问
  • 不要在不信任的设备上安装本应用

5. 数据导出与删除

您可以随时：

  • 导出所有数据为 JSON 格式备份
  • 卸载应用以删除所有本地数据
  • 通过应用内的「数据管理」功能清理数据

6. 政策更新

我们可能会不时更新本隐私政策。任何重大变更将通过应用内通知或官网公告告知用户。继续使用本应用即表示您同意修订后的政策。
